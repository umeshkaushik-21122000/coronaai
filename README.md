# Next.js Project Setup with Bun

This guide will help you set up and run your Next.js project using Bun, a modern JavaScript runtime that aims to replace Node.js.

## Prerequisites

Ensure you have the following installed on your machine:

1. **Bun:** Follow the [official Bun installation guide](https://bun.sh/docs/installation) to install Bun on your system.
2. **Git:** A version control system to manage your project code.
